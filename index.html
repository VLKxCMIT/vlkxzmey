<input type="radio" id="btn-N" onchange="sliderHeight('slide-N')" name="toggle">
заменив обе N на нужный порядковый номер
4) Добавить лейбл к кнопке ↓
<label for="btn-N"><div>Текстовка слайда</div></label>

5) В style.css добавить 100% к ширине блока (в нашем случае 7й слайд это 700%, 8й слайд 800% и т.д.) ↓
.slider-slides { ... width: 700%; ... ... }
6) Добавить новой кнопке прозрачность при нажатии → #btn-N:checked ~ .slider-controls label[for="btn-N"];
7) Осталось добавить сдвыг блока на -800px ↓
#btn-7:checked ~ .slider-inner .slider-slides { -moz-transform: translate(-4800px); /* Для Firefox */ -ms-transform: translate(-4800px); /* Для IE */ -webkit-transform: translate(-4800px); /* Для Safari, Chrome, iOS */ -o-transform: translate(-4800px); /* Для Opera */ transform: translate(-4800px); }
